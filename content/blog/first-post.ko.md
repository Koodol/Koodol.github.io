자리표시자 블로그 글입니다. 마크다운으로 작성하세요: **굵게**, *기울임*, [링크](/),
목록, 코드 블록, 인용문, 표가 모두 적절한 타이포그래피로 렌더링됩니다 — 시음 노트와
발효 일지에도 잘 어울립니다.

```python
def cheers(name: str) -> str:
    return f"주모! {name}님께 한 사발 더!"
```

> 로컬 관리자 편집기에서 Ctrl+V로 이미지를 붙여넣으면 자동으로 업로드되고,
> 사이트에서는 클릭하면 확대되는 라이트박스로 열립니다 — 배치 사진 올리기에 좋습니다.
